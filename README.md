## ✨ 它能做什么

游戏搭子每隔一段时间自动截取你的屏幕，调用阿里云百炼视觉大模型分析画面，再用 **CosyVoice 龙安欢**（欢脱元气女声）合成语音实时播报——就像一个真人朋友坐在你旁边陪你玩。

| 功能 | 说明 |
|------|------|
| 🖥️ 实时截屏分析 | 自动识别游戏名称和当前场景 |
| 🎤 有情感的语音播报 | 鼓励时激情解说，调侃时脱口秀风格 |
| 💬 五种搭子风格 | 热情鼓励 / 损友调侃 / 专业解说 / 毒舌评论 / 甜蜜女友 |
| ⚡ 立即分析 | 不等定时器，随时触发一次分析 |
| 🔇 语音开关 | 一键静音，不打扰室友 |
| 🔄 页面恢复 | 关掉浏览器再打开，自动恢复监控状态 |

## 🚀 快速开始（推荐）

### 第一步：获取阿里云百炼 API Key

1. 注册/登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)
2. 左侧菜单 →「API Key 管理」→ 创建并复制 API Key

> 画面分析（qwen-vl-max）和语音合成（cosyvoice-v3-flash）共用同一个 Key。

### 第二步：安装 Python

- **macOS**：终端运行 `brew install python`，或从 [python.org](https://www.python.org/downloads/) 下载
- **Windows**：从 [python.org](https://www.python.org/downloads/) 下载安装包，**安装时务必勾选「Add Python to PATH」**

### 第三步：一键启动

**macOS / Linux** — 打开终端，进入项目文件夹后运行：

```bash
bash setup_and_run.sh
```

**Windows** — 直接**双击** `setup_and_run.bat`

脚本会自动完成：创建虚拟环境 → 安装依赖 → 修复 SSL 证书（macOS）→ 启动服务

看到以下输出即表示启动成功：

```
✅ 浏览器访问 http://127.0.0.1:7788 开始使用
游戏搭子后端已启动：http://127.0.0.1:7788
```

### 第四步：打开界面

浏览器访问 **http://127.0.0.1:7788**，填入 API Key，选好风格，点「🚀 开始监控」！

> **以后每次启动**只需重复第三步，不会重复安装依赖。

## 🍎 macOS 额外配置：屏幕录制权限

**首次运行时**，macOS 需要授权 Python 读取屏幕：

1. 打开「系统设置」→「隐私与安全性」→「屏幕录制」
2. 点击左下角 🔒 解锁
3. 找到「终端」（Terminal）并开启

如果列表里没有终端，先运行一次脚本，系统会自动弹出授权弹窗，点允许后重新运行脚本即可。

## ❓ 常见问题

**Q：运行脚本提示「未找到 Python」？**
确认已安装 Python，Windows 用户安装时需勾选「Add Python to PATH」，安装完后重新打开命令行窗口。

**Q：macOS 截图全黑？**
前往「系统设置 → 隐私与安全性 → 屏幕录制」，确认终端已获得授权，然后重新运行脚本。

**Q：语音没有声音？**
检查系统音量，以及界面里「语音朗读」开关是否开启。如果终端有报错，检查 API Key 是否正确。

**Q：关掉浏览器后搭子还在工作吗？**
是的，后端在终端持续运行。重新打开 http://localhost:7788 会自动恢复监控状态。

**Q：想彻底停止搭子？**
在终端窗口按 `Ctrl + C`。
